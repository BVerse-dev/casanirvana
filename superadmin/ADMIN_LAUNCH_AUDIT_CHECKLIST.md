# Superadmin Launch Audit Checklist

This tracker is the execution log for the superadmin launch wiring audit. Scope is limited to visible sidebar routes in [`superadmin/src/assets/data/menu-items.ts`](/Users/andromeda/casanirvana/superadmin/src/assets/data/menu-items.ts) plus the child create/detail/edit/manage flows each route needs to function truthfully for launch.

## Status Legend

- `pending`: not audited yet
- `in_progress`: currently being audited/fixed
- `complete`: audited and aligned with launch rules
- `ok_existing`: audited and already acceptable without code changes
- `follow_up`: known issue remains outside the current wave

## Launch Rules

- No fabricated launch-facing business data.
- No admin-sensitive browser writes on audited routes.
- Aggregated or permission-sensitive reporting should use audited data contracts.
- Empty states and unavailable metrics must be explicit and truthful.
- UI structure stays intact; changes are limited to wiring, logic, truthfulness, and action behavior.
